Project 1: Cropped Letters

This project consisted of cropping letterforms using fonts Helvetica Bold, Adobe Caslon Regular, Bauer Bodoni Roman, and Clarendon Roman, to create square compositions consisting of typographic pattern and texture. Focus was placed on composition and letterform, and the balance of positive and negative space.

Project 2: Mantra Merch

This project was to create a wearable merchandise item based on a chosen personal mantra, with a strong focus on typographic design. The mantra would be communicated through the spacing and arrangement of letters. I chose to design a shoe using my mantra, as a literal representation of “walking it off.”

Project 3: Wikibook

This project was to create a book using one of Wikipedia’s longest pages. Using typographic layout to create a visual rhythm, working with grid systems, pairing typefaces, and implementing heirarchy are the main focuses. Moreover, emphasis is placed on using character and paragraph styles, master pages, page structure, and typesetting.
